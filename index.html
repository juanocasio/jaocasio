Juan Ocasio: Wannabe geek to the stars

I am an IT geek, loving all aspects of the IT world. My passion has always been programming, however I've always been a bit reluctant to search for something meaningful, so I've decided it's now or never! One of my biggest hurdles, as I’m sure it is with many of developers, has been trying to decide what programming language(s) to learn; we all seem to have sort of the shiny object syndrome. Really looking at the various technologies and digging deep, I've finally settled on a couple: JavaScript (goes without saying HTML and CSS as well) and .NET (C# using MVC). I will be rounding this out with AngularJS as my JavaScript framework. I’m still learning, but feel really good about my choices!
